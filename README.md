# FunkTyperQt
Source Code Management Utility for C++ , Java, etc.

to Compile:
make

Usage:
Run: funktyperqt
make a new directory for every project

Usage:
Keys:
F1: c++ mode
F2: java mode
h: headers, preprocessors
g: global variables
f: function
b: block
s: statements
e: end
F11: executes - g++ -c output.cpp 
F12: making of output.cpp (rename required for java)
Ctrl+c : copy
Ctrl+v : paste
Delete : delete
